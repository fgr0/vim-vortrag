
# Features - Befehle

* Grundlegende bestehen aus nur einem Buchstaben:
    - `h`, `j`, `k`, `l` : Bewegen
    - `d`                : Löschen
    - `y`                : Kopieren
    - `p`                : Einfügen
    - `/`                : Suchen
    - ...

* Kompliziertere Befehle: `:BEFEHL`
    - `:w`       : Speichern
    - `:q`       : Schließen
    - `:s`       : Ersetzen
    - `:Explore` : Öffnet einen Dateibrowser
    - ...
